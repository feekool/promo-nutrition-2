Запуск за 5 простых шагов

Не требует навыков программирования. Если умеете копировать — справитесь.

  1. Создайте аккаунт на GitHub

    Бесплатная регистрация на github.com займёт 2 минуты

  2. Откройте шаблон форума

    Перейдите по ссылке: github.com/feekool/feekool.github.io

  3. Создайте свою копию

    Нажмите зелёную кнопку "Use this template" → "Create a new repository". Дайте имя, например my-nutri-forum

  4. Включите публикацию

    В настройках репозитория: Settings → Pages → выберите ветку main → Save. Ждите ~60 секунд.

  5. Настройте под себя

    Замените тексты, логотип, цвета. Добавьте разделы «Курсы», «Отзывы», «Контакты». Ваш форум готов!

💡 Совет от практиков: Начните с минимальной версии — 2-3 раздела. Протестируйте на первых клиентах, соберите обратную связь, затем масштабируйте.
